This directory is for miscellaneous gallery images.

Examples of images to place here:
- General prosthetic photos
- Event photos
- Background and pattern images
- Infographics and charts

Recommended dimensions:
- Gallery images: 1200x800px (3:2 ratio)
- Infographics: 1200x1500px (4:5 ratio)
- Background patterns: various dimensions

Subdirectories:
- /patterns/ - For repeating background patterns and textures

Note: The prosthetic-hand.jpg file has been moved here from the root public directory
as an example of the migration process.
