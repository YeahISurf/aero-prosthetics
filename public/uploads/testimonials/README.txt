This directory is for testimonial images and avatars.

Examples of images to place here:
- Client profile photos
- Testimonial author avatars
- Before/after comparison images

Recommended dimensions:
- Avatars: 256x256px (1:1 square ratio)
- Profile photos: 512x512px (1:1 square ratio)
- Before/after images: 800x600px (4:3 ratio)

Example filenames:
- michael-johnson.jpg
- sarah-williams.jpg
- before-after-johnson.jpg
